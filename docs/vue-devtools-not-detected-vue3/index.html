    [Vue3] Vue devtools - not detected

    2021-11-12

    • Vue3
    • Vue devtools

    작성 시점 기준으로 Vue.js devtools가 Vue3 프로젝트를 감지하지 못하는 이슈가 있습니다.
    베타 버전의 devtools를 새로 다운로드해서 사용해야 하고, 베타 툴이 안정적으로 동작하기 위해서는 기존 devtools는 비활성화 해놓아야 합니다.

    Vue.js devtools(beta) 다운로드 페이지
